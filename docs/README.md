# Document of gcrack

## Dependencies
We use sphinx to automatically generate the documentation.
Install the dependencies.
```shell
conda activate gcrack
conda install sphinx sphinx-rtd-theme
```

## How to compile the doc

To generate the doc in html:
```shell
make html
```
The doc can then be opened using in build/index.html.




# Documentation of gcrack

## Dependencies
We use Sphinx to automatically generate the documentation.

Install the dependencies:
```shell
conda activate gcrack
conda install sphinx sphinx_rtd_theme
```

## How to compile the doc

To generate the documentation in HTML format:
```shell
make html
```
The documentation can then be opened using a web browser by navigating to `build/html/index.html`.
