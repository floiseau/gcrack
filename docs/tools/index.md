# Overview

When you install the `gcrack` package, several additional utility scripts are installed.
These scripts complement `gcrack` by providing features to analyze the simulations.
Currently, the provided tools are :

- **[`gcrack-wulff-plotter`](display_wulff_diagram.md)**
  This script exports the displacement field (and the deformed configuration) from a `gcrack` simulation as image files.
  It can be used to generate video animation of the simulation.

- **[`gcrack-displacement-plotter`](display_displacement.md)**
  This script exports the Wulff diagram from a `gcrack` simulation as image files.

For more details on each tool, check the provided links.
